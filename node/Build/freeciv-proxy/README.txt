Freeciv web proxy

Start the Freeciv HTML5 WebSocket proxy like this:
python freeciv-proxy.py

See status on this URL when proxy is running: 
http://localhost:8002/status

The proxy requires Python 3.3.

The HTML5 Websocket server requires:
- Tornado 3.0.1 - http://www.tornadoweb.org/
